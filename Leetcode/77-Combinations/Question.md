# 77. Combinations

[Original Page](https://leetcode.com/problems/combinations/)

Given two integers _n_ and _k_, return all possible combinations of _k_ numbers out of 1 ... _n_.

For example,  
If _n_ = 4 and _k_ = 2, a solution is:

<pre>[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
</pre>
